App Permissions Explained

Command Helper requires certain permissions to provide its full functionality. Below is an explanation of each permission we request and why it's needed:

File System Access

Command Helper requires file system access to:

  • Import script files via drag-and-drop (Pro feature)
  • Export directories and commands (Pro feature)
  • Save command output to files when requested

We only access files you explicitly select or locations you specifically grant permission to.

Notifications

If enabled, Command Helper uses notifications to:

  • Alert you when commands copied to your clipboard.

Notifications are optional and can be disabled in system settings.

Network Access

Command Helper requires internet access for:

  • Processing subscription purchases

Your commands never leave your device unless you explicitly enable the backup feature.

Permission Management

You can review and modify permissions granted to Command Helper at any time:

  1. Open System Preferences/Settings on your Mac
  2. Navigate to Security & Privacy → Privacy
  3. Select the relevant category (Files and Folders, Notifications, etc.)
  4. Find Command Helper in the list and adjust permissions as needed
